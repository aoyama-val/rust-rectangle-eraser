# rust-rectangle-eraser

A Game similar to Quarth (a.k.a Block Hole).


## Requirement

- SDL 2
- Rust


## Controls

```
Left, Right : Move player
Up          : Scroll
Space       : Shoot
Enter       : Restart when gameover
```
